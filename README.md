# 🛍️ Mini Shop

A minimalist e-commerce platform built with **React 19** and **Vite**. This project focuses on demonstrating modern frontend development practices, specifically emphasizing type-safe state management, responsive UI, and seamless API integration.

## ⚡ Features
Users can:
- Search Products: Real-time search functionality across the entire product catalog.
- Manage Cart: Add/remove products and adjust quantities with persistent state.
- Explore Collections: Dynamic category pages featuring live thumbnails and filtering.
- Fully Responsive: A mobile-first design that ensures a flawless experience on all devices.

## 🛠️ Tech Stack
- React 19 – Core UI library for building a fast and interactive interface.
- TypeScript – Used throughout the project for static typing, ensuring code reliability and error prevention.
- Redux Toolkit – Efficient global state management for the shopping cart and search.
- Tailwind CSS – Utility-first, minimalist, and responsive styling.
- Axios – Handling asynchronous data fetching from the DummyJSON API.
- Lucide React – Clean and consistent iconography.
